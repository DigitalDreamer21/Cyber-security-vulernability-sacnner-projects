# 🔐 Cyber Security Scanner Pro

A GUI-based cybersecurity tool developed using Python to scan websites for common vulnerabilities and generate professional reports.

---

## 🚀 Features
- 🔍 Multi-threaded port scanning
- 🔐 Security header analysis
- 💉 SQL Injection detection
- ⚡ XSS vulnerability testing
- 🖥️ GUI interface using Tkinter
- 📄 Automated PDF report generation

---

## 🛠️ Tech Stack
- Python
- Tkinter (GUI)
- Requests
- ReportLab
